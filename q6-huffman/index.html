Huffman Coding

Build a Huffman tree for symbols A, B, C, D

Problem Statement

Construct a Huffman code for the source below. Show the merging steps or tree, assign binary codewords, compute the average code length, and compare it with a fixed-length binary code for four symbols.

SymbolABCD
Frequency8422
Key Concepts
  • Huffman coding — greedy entropy coding; assigns shorter codewords to more frequent symbols.
  • Prefix-free — no codeword is a prefix of another, enabling unambiguous decoding.
  • Average length — Σ (probability × code length) over all symbols.
  • Fixed-length code — for N symbols needs ⌈log₂ N⌉ bits; here N=4 → 2 bits.
