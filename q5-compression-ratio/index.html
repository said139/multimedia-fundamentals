Compression Ratio

JPEG compression ratio for a 1920×1080 RGB image

Problem Statement

A raw RGB image has resolution 1920 × 1080, with 8 bits per colour channel. It is compressed into a 1.2 MB JPEG file. What is the approximate compression ratio?

Key Concepts
  • Raw RGB — 3 channels (R, G, B), 8 bits each → 24 bits per pixel total.
  • Compression ratio — ratio of original (uncompressed) size to compressed size.
  • JPEG — lossy compression standard; uses DCT + quantization + Huffman coding.
  • 1 MB = 1,048,576 bytes
Step-by-Step Solution

Total pixels: 1920 × 1080 = 2,073,600 pixels

Bits per pixel (RGB, 8-bit): 3 channels × 8 = 24 bits/pixel

Raw size in bits: 2,073,600 × 24 = 49,766,400 bits

Raw size in bytes: 49,766,400 ÷ 8 = 6,220,800 bytes

Raw size in MB: 6,220,800 ÷ 1,048,576 ≈ 5.93 MB

JPEG size: 1.2 MB

Compression ratio: 5.93 ÷ 1.2 ≈ 4.94 : 1

Raw size = 1920 × 1080 × 3 bytes = 6,220,800 bytes ≈ 5.93 MB Ratio = 5.93 MB ÷ 1.2 MB ≈ 4.94 : 1 (~5:1)
✓ Compression Ratio ≈ 4.94 : 1 (≈ 5 : 1)
