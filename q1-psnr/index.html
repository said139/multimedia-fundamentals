Question 1 — 4 Points

PSNR Calculation

Peak Signal-to-Noise Ratio for a reconstructed image

Problem Statement

A reconstructed image has a mean squared error MSE = 100 compared with the original 8-bit image. What is the approximate PSNR?

PSNR = 10 × log₁₀( 255² / MSE )
Key Concepts
  • MSE — Mean Squared Error: average of squared pixel-level differences between original and reconstructed image.
  • MAX value — For an 8-bit image, the maximum pixel intensity is 255.
  • PSNR — Higher values indicate better quality; a PSNR above ~30 dB is generally considered acceptable.
Step-by-Step Solution

Identify the MAX value for an 8-bit image: MAX = 255

Square the MAX value: 255² = 65,025

Divide by MSE: 65,025 ÷ 100 = 650.25

Apply log₁₀: log₁₀(650.25) ≈ 2.8130

Multiply by 10: 10 × 2.8130 = 28.13 dB

PSNR = 10 × log₁₀( 65025 / 100 ) = 10 × 2.8130 ≈ 28.13 dB
✓ PSNR ≈ 28.13 dB
Interpretation
PSNR RangeQuality
> 40 dBExcellent — virtually indistinguishable from original
30 – 40 dBGood — minor artifacts, acceptable for most uses
≈ 28 dB ← our resultFair — visible distortion, adequate for some applications
