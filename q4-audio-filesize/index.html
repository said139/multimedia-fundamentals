Question 4 — 4 Points

Audio File Size

Storage for a 5-second stereo PCM recording

Problem Statement

What is the file size of a 5-second audio recording sampled at 22 kHz, 16-bit, stereo?

Key Concepts
  • Sample rate (22 kHz) — 22,000 audio samples captured per second per channel.
  • Bit depth (16-bit) — precision of each sample; 16 bits = 65,536 possible amplitude levels.
  • Stereo — two independent channels (Left + Right), so multiply by 2.
  • PCM — Pulse Code Modulation; the raw, uncompressed audio format (like WAV).
Step-by-Step Solution

Samples per second per channel: 22,000

Channels: 2 (stereo)

Total samples per second: 22,000 × 2 = 44,000

Total samples for 5 sec: 44,000 × 5 = 220,000

Total bits: 220,000 × 16 = 3,520,000 bits

Convert to bytes: 3,520,000 ÷ 8 = 440,000 bytes

Convert to KB: 440,000 ÷ 1,024 ≈ 429.69 KB ≈ 430 KB

Size = 22,000 × 16 × 2 × 5 = 3,520,000 bits = 440,000 bytes ≈ 430 KB
✓ File Size ≈ 440,000 bytes (~430 KB)
