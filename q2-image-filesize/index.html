Image File Size

Uncompressed storage for a 512×512 RGB image

Problem Statement

A 512 × 512 RGB image is stored uncompressed with 24 bits per pixel. What is the file size?

Key Concepts
  • 24 bits/pixel (RGB) — 8 bits each for Red, Green, and Blue channels.
  • Uncompressed — every pixel stored at full bit depth, no encoding overhead.
  • 1 byte = 8 bits  |  1 KB = 1,024 bytes  |  1 MB = 1,048,576 bytes
Step-by-Step Solution

Total pixels: 512 × 512 = 262,144 pixels

Total bits: 262,144 × 24 = 6,291,456 bits

Convert to bytes: 6,291,456 ÷ 8 = 786,432 bytes

Convert to KB: 786,432 ÷ 1,024 = 768 KB

Convert to MB: 768 ÷ 1,024 = 0.75 MB

File size = 512 × 512 × 24 bits = 6,291,456 bits = 786,432 bytes = 768 KB = 0.75 MB
✓ File Size = 768 KB (0.75 MB)
